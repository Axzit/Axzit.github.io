# Axzit

This repository is now a modern React web app scaffold for Axzit.

## Getting started
This site is built to deploy directly from the repository root on GitHub Pages.

1. Open `index.html` in the repository root.
2. If you want to run a local development server, use any static file server.

## About
Axzit is a futuristic portfolio and brand presence built with React-style browser modules.
The page is served from the repository root so the GitHub Pages user site works correctly.
